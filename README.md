# 口算工坊（math-sheet）

Family Apps 中的纯静态小应用：按年级选题型、混合组卷，一键生成 A4 口算练习和参考答案页，
卷号可复现重印。

## 功能（v0.3 重写版）

- **17 种题型按年级分组**（一/二/三年级），覆盖进位/退位/不进位/不退位、表内乘除、
  连加连减、乘加乘减、有余数除法、整十整百口算；可多选，均匀混合、随机穿插组卷。
- **四种出题形式**：标准（`8 + 7 = ▁`）、求未知数（`8 + □ = 15`）、比大小（`8 + 7 ○ 16`）、混合。
- **自动排版**：选字号（小/中/大）、行距（紧凑/标准/宽松）、列数（2/3/4），每页题数自动计算，
  从根本上保证打印不溢出；宽算式/大字号自动限制列数。
- **卷号可复现**：卷号印在页脚，同一卷号 + 同一设置可重印同一份卷子（给二娃/重做）。
- **同卷去重**、题号开关、答题横线开关、紧凑答案页（一页答案覆盖多页题目）。
- 配置自动记忆（localStorage）；`?s=卷号` 链接直达。
